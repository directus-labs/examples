## ðŸš€ Getting Started

1. Clone this repo.

2. Install dependencies for this example.

   ```shell
   cd gatsby
   npm install
   ```

3. Create a `.env` file by copying the provided `.env.example` file.

4. Start the development server.

   ```shell
   npm run develop
   ```

   Your Directus Gatsby example is now running at <http://localhost:8000>.
